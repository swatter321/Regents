Exam-day game plan

  • Use the Reference Tables booklet constantly. Heat formulas → Table T, half-lives → Table N, polyatomic ions → Table E, solubility curves → Table G.
  • Watch units. Gas-law temperatures must be in Kelvin (K = °C + 273). STP is 273 K and 101.3 kPa.
  • Read fully — "which is not" and "least" trip people up. Underline the key word.
  • Never leave a multiple-choice blank. No penalty for guessing — eliminate two and pick.
  • Bring a 4-function calculator and check it works the night before.
